## 👨‍💻 How to Setup

```bash
    # Clone the project
    $ git clone https://github.com/gabrielteixeira-0814/twitter_clone.git
```

```bash
    # Enter directory
    $ cd  C:\Projetos\twitter_clone\public
```

```bash
    # This command will start a development server at http://localhost:8080
    $ php -S localhost:8080
```
